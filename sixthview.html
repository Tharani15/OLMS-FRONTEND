Course Overview

This course is designed to build a strong foundation in data structures and algorithms using Java. It focuses on writing efficient, optimized code for solving real-world problems. Students will learn how to use Java collections, implement custom data structures, and apply algorithmic techniques like searching, sorting, and recursion. This course aims to enhance problem-solving skills and prepare learners for coding interviews and competitive programming.

Objectives

  • Understand and implement essential data structures in Java
  • Learn algorithmic strategies such as recursion, sorting, and searching
  • Use Java Collections Framework effectively
  • Analyze time and space complexity of algorithms
  • Solve real-world problems using the right data structures
  • Prepare for technical interviews with algorithmic thinking

Syllabus Includes

  • Introduction to Data Structures and Algorithms
  • Arrays, Strings, and Linked Lists (Singly & Doubly)
  • Stacks, Queues, and Priority Queues
  • Recursion and Backtracking Techniques
  • Searching Algorithms: Linear Search, Binary Search
  • Sorting Algorithms: Bubble, Selection, Insertion, Merge, Quick Sort
  • Hashing and HashMaps
  • Trees and Binary Search Trees (BST)
  • Graphs, BFS and DFS Traversals
  • Time and Space Complexity Analysis (Big-O Notation)
